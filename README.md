# Håndball Statistikk App

En komplett app for å føre statistikk over spillerprestasjoner i håndball.

## Innhold
- `index.html` - Hoveddokumentet
- `styles.css` - All styling
- `app.js` - All funksjonalitet

## Hvordan publisere på Netlify

### Metode 1: Drag & Drop (Enklest)
1. Gå til [netlify.com](https://www.netlify.com)
2. Registrer deg (gratis)
3. Dra hele mappen "handball-app" inn i Netlify
4. Vent 5-10 sekunder
5. Ferdig! Du får en URL som: `https://ditt-navn.netlify.app`

### Metode 2: Netlify Drop
1. Gå til [app.netlify.com/drop](https://app.netlify.com/drop)
2. Dra hele mappen "handball-app" inn på siden
3. Ferdig! Får URL umiddelbart

## Bruke på iPad
1. Åpne URL-en i Safari
2. Trykk "Del"-knappen (firkant med pil opp)
3. Velg "Legg til på Hjem-skjerm"
4. Nå har du en app-ikon!

## Gjøre endringer
1. Last opp filene til Claude
2. Beskriv hva du vil endre
3. Last ned oppdaterte filer
4. Erstatt de gamle filene på Netlify

## Funksjoner
- ✅ Registrer skudd med nøyaktige koordinater
- ✅ Mål/redning/utenfor
- ✅ Keeper-statistikk
- ✅ Tekniske feil
- ✅ Statistikk per omgang og totalt
- ✅ Visualisering av skudd
- ✅ Eksport til JSON
